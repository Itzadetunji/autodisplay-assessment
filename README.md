## Getting Started Locally

How to start the main project on your local system

### Insall Dependencies

```bash
npm install
# or
bun install
```

### Run Project

```bash
npm run dev
# or
bun dev
```

Open [http://localhost:6010](http://localhost:6010) with your browser to see the result.

## Testing

I had some extra time and decided to add tests to the api endpoints using jest

```bash
# Run all tests
bun run test

# Run tests with coverage
bun run test:coverage
```
